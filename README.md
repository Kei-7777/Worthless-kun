# Worthless-kun

## 動作環境
- Paper 1.15.2 (387)
- Windows 10 Pro 64bit

## 前提プラグイン
- SkinsRestorer

## 使い方
- ゲーム内で/wl <プレイヤー名>で黒いスキン切り替え。
- コンフィグでベースにする黒いスキンのプレイヤーを選択。
(黒いスキンのプレイヤーはWorthless\config.ymlを参考に。)
